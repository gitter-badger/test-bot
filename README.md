# hubot-wink

NBC for messages

See [`src/wink.coffee`](src/wink.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-wink --save`

Then add **hubot-wink** to your `external-scripts.json`:

```json
[
  "hubot-wink"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
